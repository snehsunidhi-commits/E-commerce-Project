# E-commerce-Project

Developed HarryShop, a comprehensive e-commerce database project for a coder merchandise store using SQL. Designed and implemented a relational database schema with multiple interconnected tables, establishing data integrity through primary and foreign key relationships. Populated the database with realistic business data, including customers, products, orders, and transactions. Wrote complex SQL queries involving joins, subqueries, aggregate functions, and filtering techniques to analyze sales performance, customer purchasing behavior, product demand, and revenue trends. Generated actionable business insights through data exploration and reporting, demonstrating practical applications of database management and data analytics in an e-commerce environment.
