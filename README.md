# E-commerce Database Analysis - SQL

Developed HarryShop, a SQL-based e-commerce database project for a coder merchandise store. Designed and implemented a normalized relational database with multiple interconnected tables using primary and foreign key relationships to ensure data integrity and efficient data management. Populated the database with realistic business data, including customers, products, orders, and transactions. Performed advanced SQL analysis using joins, subqueries, aggregate functions, CTEs, and filtering techniques to evaluate sales performance, customer purchasing behavior, product demand, and revenue trends. Generated business insights through data-driven reporting to support strategic decision-making and improve overall store performance.

Key Skills: SQL, Database Design, Relational Databases, Data Analysis, Joins, Subqueries, CTEs, Aggregate Functions, Data Modeling, Business Intelligence.
